# LINXAR | O Motor de Inteligência para E-commerce

Site institucional da **LINXAR**, uma plataforma de IA especializada em padronização e enriquecimento de dados de produtos para marketplaces.

![LINXAR Preview](https://via.placeholder.com/1200x630/020202/FFFFFF?text=LINXAR+AI+ENGINE)

## 🚀 Tecnologias

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Animações**: [Framer Motion](https://www.framer.com/motion/)
- **Linguagem**: [TypeScript](https://www.typescriptlang.org/)
- **Deploy**: [Vercel](https://vercel.com/)

---

## ✨ Diferenciais

- **Arquitetura High-Performance**: Design minimalista e premium (Apple-like).
- **Interface Glassmorphism**: Uso de transparências e borrões inteligentes para um look de elite.
- **Narrativa Privada (NDA-Ready)**: Comunicação focada em escala e segurança industrial.
- **Micro-animações**: UX fluida baseada em estados de movimento.

---

## 🛠️ Instalação e Uso

1. **Clonar repositório**:
   ```bash
   git clone https://github.com/linxar/site-institucional.git
   ```

2. **Instalar dependências**:
   ```bash
   npm install
   ```

3. **Rodar em desenvolvimento**:
   ```bash
   npm run dev
   ```

4. **Build para produção**:
   ```bash
   npm run build
   ```

## ☁️ Deploy na Vercel

O deploy da aplicação é feito automaticamente via integração com o **GitHub**. Para novos ambientes:

1. Importe o repositório na Vercel.
2. Defina as variáveis de ambiente (se houver) no Dashboard da Vercel.
3. O build é otimizado automaticamente para o Next.js 14.
