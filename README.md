# LINXAR | O Motor de Inteligência para Marketplaces

Site institucional da **LINXAR**, uma plataforma de IA especializada em padronização, enriquecimento e otimização de anúncios para marketplaces.

![LINXAR Preview](https://via.placeholder.com/1200x630/020202/FFFFFF?text=LINXAR+AI+ENGINE)

## 🚀 Tecnologias

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Animações**: [Framer Motion](https://www.framer.com/motion/)
- **Linguagem**: [TypeScript](https://www.typescriptlang.org/)
- **Deploy**: [Vercel](https://vercel.com/)

---

## ✨ Diferenciais Imersivos

- **Aurora 3D System**: Fundo dinâmico com sistema de feixes de aurora boreal multicamadas (Profundidade, Marca e Destaque).
- **Marketplace Ticker**: Carrossel infinito com as principais conexões do mercado (Curva A e B).
- **Glassmorphism de Elite**: Interface translúcida com desfoques inteligentes e granulação orgânica (Noise texture).
- **Institucional Completo**: Página "Quem Somos" integrada ao ecossistema de inovação Sprint (UTFPR).
- **Arquitetura High-Performance**: Design minimalista e premium inspirado em padrões globais de UX.

---

## 🛠️ Instalação e Uso

1. **Clonar repositório**:
   ```bash
   git clone https://github.com/trueluuck/linxar-site.git
   ```

2. **Instalar dependências**:
   ```bash
   npm install
   ```

3. **Rodar em desenvolvimento**:
   ```bash
   npm run dev
   ```

4. **Build para produção**:
   ```bash
   npm run build
   ```

---

## ☁️ Deploy na Vercel

O deploy da aplicação é feito via GitHub at [linxar.com.br](https://linxar.com.br).

1. Importação automática via branch `main`.
2. Build otimizado para produção com Next.js 14.
3. Certificado SSL e CDN global ativa.
